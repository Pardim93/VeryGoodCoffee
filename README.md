## VeryGoodCoffee

Find more about the best coffee drinks and save the best as your favorites

## Instructions

Open the `VeryGoodCoffee.xcodeproj` file in Xcode and run it in a simulator or your own device!

## Application

- Get recommended a coffee cup image drink 
- Decide if you want to set it as one of your favorites or get the next cup!
- Go to favorites to check out for your saved pics
- Unmark the heart on the pictures you want to remove 

## Coding 
- Xcode 26
- iOS 26
- SwiftUI + MVVM + Clean + Testing + Logging
